# Roblox username sniper!
Makes and finds valid Roblox username with a specified length and logs it out to a file.
<br>
<br>
`make_usernames.py` Creates all possible combinations of Roblox usernames given a username length, outputs to text files in 17MB chunks (all can be modified).

## WARNING!
`make_usernames.py` will output text files of considerable size, make sure you have adequate space!

<img style="width:50%" src="https://user-images.githubusercontent.com/86912923/222378535-fc9033b2-6e76-4360-bc56-4360e26425e5.png"/>

Default output is not_taken.txt. <br>
Python2 is not supported and will not work. <br>

This works on replit.

# Installing required packages:
```
pip install requests
```

# Running the script:
```
python3 main.py
```

# Run multiple instances
```
sh auto.sh
```
